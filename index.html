--- layout: page title: Lehigh Outfitters Style Guide description: Design standards that comply with ADA for the web --- # Lehigh Style Guide The Lehigh style guide are resources to help create and standardize principals for the Lehigh brands, which include Lehigh Outfitters, Lehigh Safety Shoes, CustomFit, and SlipGrips. It's to be used for both internal and external resources to help create brand consistency and strive to be ADA compliant on the web. Having a consistent brand enables a better user experience, performance, and a more accessible web which directly impacts overall revenue growth. Erratic design patterns or improper implementations can cause havoc to properties with multiple people working on them. ## Brands ### Lehigh Outfitters Occupational footwear for any environment. ### Lehigh Safety Shoes Protective and safety shoes and boots. ### CustomFit The leader in managed safety footwear programs. ### SlipGrips Slip-resistant shoes for restaurants, hospitals, airlines, vets, schools, etc. ## Contribution This document was written by Cody Saylor (cody.saylor@lehighoutfitters.com) based on [the style guide guide](https://github.com/bradfrost/style-guide-guide) by [Brad Frost](https://github.com/bradfrost).
