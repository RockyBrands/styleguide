--- layout: page title: Rocky Brands Style Guide description: Design standards that comply with ADA for the web --- # Rocky Brands Style Guide These style guides are resources to help create and standardize principals for the Rocky brands, which include Rocky Boots, Durango Boots, Georgia Boots, Lehigh Outfitters, Lehigh Safety Shoes, CustomFit, and SlipGrips. It's to be used for both internal and external resources to help create brand consistency and strive to be ADA compliant on the web. Having a consistent brand enables a better user experience, performance, and a more accessible web which directly impacts overall revenue growth.
